FUNCIONAMIENTO:
Primero: se debe dar click en algun sitio del mapa que está ubicado en la ventana principal.
Al dar click se creará un nodo, este será el nodo principal (el nodo en el cual estará ubicado JSDomicilios)
partiendo de este nodo se pueden crear los demás y crear relaciones entre ellos.
Para crear una arista entre un nodo y otro, se deberá dar click en ambos nodos(estos se tornaran de color amarillo al ser seleccionados).
Luego de seleccionarlos aparecerá una ventana que le pedirá la distancia entre estos dos nodos (en kilómetros). 

El botón start, funciona para buscar el camino mínimo entre los 2 nodos que se le ingresen. 

Se pueden crear múltiples nodos y aristas.
